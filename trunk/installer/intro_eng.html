Plantlore

Introduction

Plantlore is a database system to be used for occurrence data management. It is a client-server application written in Java and developed under the GPL license. Plantlore supports local and remote work via a network. This installer installs the Plantlore rich client and Plantlore server. The project also includes a web client which you have to install separately.

Feature highlights

  • Simple and fast occurrence adding and editation (the checklist, auto completition, ...)
  • User, author and project management
  • Import and export of ABCD schema and DarwinCore compatible XML
  • Localization - English and Czech for the moment
  • Scheda printing
  • Open sourced under the GPL license
  • ... and more

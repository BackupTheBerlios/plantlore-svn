Plantlore

Úvod

Plantlore je databázový systém pro zadávání a vyhledávání dat o výskytu rostlin šířený pod licencí GPL. Je navržen jako aplikace s architekturou klient server. Umožňuje jak práci lokální tak i vzdálenou po síti. Tento instalátor instaluje Plantlore klienta a server napsané v programovacím jazyce Java. Součástí projektu Plantlore je i webový klient, který se instaluje zvlášť.

Vybrané funkce

  • Rychlé zadáváni a editace nálezù ("škrták", automatické doplňování, ...)

  • Správa dat o uživatelích, autorech a projektech

  • Import a export do XML kompatibilního se standardy ABCD schema a DarwinCore

  • Lokalizace - zatím čeština a angličtina

  • Tisk sched

  • Volně dostupné zdrojové kódy pod licencí GPL

  • ... a další

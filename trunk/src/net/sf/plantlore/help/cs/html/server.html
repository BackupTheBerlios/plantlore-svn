Spouštění serveru

Aby se mohli k Vaší databázi připojovat i jiní uživatelé z počítačů v síti, musí být spuštěn server. Server zprostředkovává připojení vzdálených uživatelů

1. Start serveru/Připojení k serveru

Server lze spustit pouze na lokálním počítači na vybraném portu. Nezadáte-li port, využije se standardní port 1099. Musí být zaškrtnutá volba "Vytvořit nový server a spustit jej". Aby server nemohl ukončit někdo nepovolaný, je možné nastavit heslo, kterým bude server chráněn. Budete-li se chtít připojit k tomuto serveru později jako jeho administrátor, budete se muset prokázat tímto heslem.

Víte-li, že někde již běží server, který jste spustili, můžete jej vzdáleně ovládat. K tomu stačí vybrat volbu "Připojit se k existujícímu serveru", zadat jméno počítače a port. Port můžete vynechat, pak se použije standardní port 1099. Současně se musíte prokázat heslem administrátora serveru, které bylo zadáno při jeho spouštění.


2. Ovládání serveru

Po připojení k serveru jej můžete ovládat.

  • Obnovit. Obnoví seznam s uživateli přihlášenými k tomuto serveru.
  • Odpojit uživatele. Odpojí všechny vybrané (označené) uživatele od serveru (tzv. kick).
  • Vypnout server. Ukončí činnost serveru, všichni uživatelé přihlášení k tomuto serveru budou odpojeni bez ohledu na to, zda mají spuštěny nějaké operace.
